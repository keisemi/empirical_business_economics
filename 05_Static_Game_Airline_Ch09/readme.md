- 本章のデータはSugawara and Omori (2012)のデータを使用しています。本データは菅原慎矢さんのご厚意により使用させていただいております。
- `data`フォルダには、`id_port_dyad_long.csv` のみが入っています。
- 他に必要な`x_ANA.csv`, `x_JAL.csv`, `y.csv` については、以下の菅原さんのwebsiteかDropboxからダウンロードして、`data`フォルダに保存してください。
  - https://sites.google.com/view/shinya-sugawara/home
