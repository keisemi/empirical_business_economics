

# トピック 02: 集計データによる離散選択モデルの推定及び価格付けシミュレーション (第3-5章)

## フォルダ構成

- `data`: 元データが入っています。
- `intermediate`: プログラムを回す中で作成される中間ファイル(クリーニングしたデータなど)の保管場所
- `output`: 原稿に掲載する図(画像)ファイルや、表ファイル(texやcsv)を保存する場所です。

## プログラム

- `main_ch03.R`, `main_ch04.R`, `main_ch05.R`: 各章のメインプログラムです。


## 計算時間 (デフォルト環境)

- 3章: 1分以内
- 4章: 7分程度
- 5章: 10分程度
