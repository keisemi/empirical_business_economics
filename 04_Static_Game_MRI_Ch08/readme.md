


# トピック 04: 完備情報の静学ゲームの推定 (第8章)

## フォルダ構成

- `data`: 元データが入っています。
- `output`: 原稿に掲載する図(画像)ファイルや、表ファイル(texやcsv)を保存する場所です。

## プログラム

- `main_Ch08_01_BR1991.R`: Bresnahan and Reiss (1991)モデルのメインプログラムです。
- `main_Ch08_02_Berry1992.R`: Berry (1992)モデルのメインプログラムです。


## 計算時間 (デフォルト環境)

- `main_Ch08_01_BR1991.R`: 1分以内
- `main_Ch08_02_Berry1992.R`: ワークステーション環境で並列計算(10コア)で7時間程度かかりました。なお、反実仮想シミュレーションは15分程度でした。
